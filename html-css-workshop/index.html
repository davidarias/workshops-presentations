The three parts of a web page

  • HTML: content and structure
  • CSS: format and style
  • JavaScript: user interaction

HTML

  • It means Hypertext Markup Language
  • It is a text file that desbribes the content and structire of a web page.

My first web page


<!doctype html>
<html>
    <head>
        <meta charset="utf-8">
        <title>My first web</title>
    </head>
    <body>
        <h1>header</h1>
        <p>A paragraph.</p>
    </body>
</html>

Copy this code in your editor and see the result

The display CSS property

Determine if there is a new line break after the HTML element

  • Inline: there is no break
  • block: there is a break

Elements have a default value for the display property

  • inline: a
  • block: p, h1, h2...,

Two new HTML elements

  • <div> </div> ( block by default )
  • <span> </span> ( inline by default )

This elements are used to divide the page in diferent sections with diferent styles

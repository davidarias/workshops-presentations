Introduction to jQuery

  • jQuery is a JS library
  • A library is a group of functions that helps us to do some task
  • jQuery will help us to interact with the web page

Adding the jQuery library to your web page


<script src="https://code.jquery.com/jquery-2.2.2.min.js">
</script>

jQuery Selectors

Similar to css selectors


// this function gets all the paragraphs in the web page
$( "p" );
// this gets all the elements with the class warning
$( ".warning" );
// this gets the element with the unique indentifier title
$( "#title" );

Some HTML interaction


// hide all the paragraphs in the page
$( "p" ).hide();
// show all the elements with warning class if hidden
$( ".warning" ).show() ;
// change the value of the element with id title
$( "#title" ).html("my new title");

Events

When a user interact with the web page, a event is fired. We can catch the event and execute some action


// function that sets a background of an element to red
function turnRed(){
    $(this).css("background", "red");
}

// when the user click on the element with id myelement
// the turnRed function is called, and the element background
// is chaged to red
$( "#myelement" ).on("click", turnRed);

Magic!
